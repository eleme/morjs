快速上手

环境准备​

首先需要有 node, 并确保 node 版本为 14 或以上。（推荐用 nvm 来管理 node 版本，windows 下推荐用 nvm-windows）

mac 或 linux 下安装 nvm。

$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
$ nvm -v
0.39.1

安装 node。

$ nvm install 14
$ nvm use 14
$ node -v
v14.19.0

然后需要包管理工具。node 默认包含 npm

创建项目​

先找个地方建个空目录。

mkdir myapp && cd myapp

全局安装 mor cli 工具。

npm i @morjs/cli -g

通过 mor cli 工具创建项目：

mor init

然后你会看到如下界面：
